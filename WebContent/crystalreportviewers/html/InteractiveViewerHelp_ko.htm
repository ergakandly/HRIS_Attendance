대화형 뷰어 도움말

대화형 뷰어를 사용하면 보고서의 데이터를 볼 수 있으며 보고서 데이터에 대해 복잡한 검색 작업도 수행할 수 있습니다. 마우스/커서를 뷰어의 왼쪽으로 이동하면 그룹 트리를 탐색할 수 있는 보조 메뉴가 나타납니다. 뷰어에서는 보고서 데이터에 대해 텍스트 및 부울 검색 작업을 수행할 수도 있습니다. 이 메뉴가 항상 표시되도록 "고정"시킬 수 있습니다.

다음은 이 도움말에 포함된 전체 항목 목록입니다.

  • 그룹 트리
  • 텍스트 검색
  • 부울 검색
  • 내보내기
  • 확대/축소
  • 새로 고침
  • 인쇄
  • 페이지별로 탐색
  • 검색 결과 테이블

그룹 트리

그룹 트리에는 보고서 데이터가 그룹화되는 값 목록이 제공됩니다. 그룹 트리 단추를 클릭하면 그룹 트리가 표시되거나 숨겨집니다. 보고서의 특정 그룹을 탐색하려면 그룹 트리의 그룹 목록에서 해당 그룹을 클릭하십시오.

자세한 내용은 Crystal Reports 온라인 도움말(crw.chm)에서 "그룹화" 항목을 참조하십시오.

항목 목록으로

텍스트 검색

보고서에서 임의의 텍스트 문자열을 검색할 수 있습니다. 텍스트 입력 필드에 문자열을 입력하고 입력란 옆의 찾기 단추를 클릭하십시오.

참고: 현재 위치가 2페이지인 경우 보고서의 1페이지에 있는 내용을 검색하면 해당 항목을 찾을 수 없습니다. 뷰어는 다음 페이지부터 앞으로만 검색할 수 있으므로 현재 위치가 3페이지인 경우 보고서의 4페이지부터 끝까지만 검색합니다.

항목 목록으로

부울 검색

참고: 경우에 따라 뷰어에 부울 검색 기능이 없을 수도 있고 검색 문을 볼 수는 있지만 수정할 수 없을 수도 있습니다. 여기에서 수행할 수 있는 작업은 관리자가 보고서를 설정한 방식에 따라 다릅니다.

부울 검색 기능을 사용하면 특정 기준에 따라 보고서 데이터의 하위 집합을 표시할 수 있습니다.

다음과 같은 두 단계로 보고서에 대한 부울 검색 문을 지정합니다.

단계 1: 검색 기준을 입력합니다.

  1. 필드를 선택합니다.
  2. 비교 연산자 또는 부울 연산자를 선택합니다.
  3. 입력란에 값을 입력하여 식을 완성합니다.
  4. 지팡이 모양의 단추를 클릭합니다.

단계 2: 검색 테이블에 표시할 필드를 선택하고 지팡이 모양의 단추를 다시 클릭합니다.

부울 검색 결과를 나타내는 검색 테이블이 보고서에 표시됩니다. 입력한 기준에 일치하는 정보가 없으면 검색 테이블이 표시되지 않습니다.

항목 목록으로

내보내기

보고서를 내보내려면 내보내기 단추를 클릭합니다.

Crystal Reports™(RPT)로 내보내기

내보내기 페이지의 드롭다운 목록에서 Crystal Reports (RPT)를 선택하고 내보내기 단추를 클릭하면 새 보고서를 저장하거나 열 수 있습니다.

Adobe Acrobat™(PDF)으로 내보내기

내보내기 페이지의 드롭다운 목록에서 Acrobat 형식 (PDF)을 선택하고 내보내기 단추를 클릭하면 PDF를 저장하거나 열 수 있습니다.

Microsoft Word™(DOC)로 내보내기

내보내기 페이지의 드롭다운 목록에서 MS Word를 선택하고 내보내기 단추를 클릭하면 새 문서를 저장하거나 열 수 있습니다.

Microsoft Excel™(XLS)로 내보내기

내보내기 페이지의 드롭다운 목록에서 MS Excel 97-2000을 선택하고 내보내기 단추를 클릭하면 새 스프레드시트를 저장하거나 열 수 있습니다.

Microsoft Excel™(데이터만)(XLS)로 내보내기

내보내기 페이지의 드롭다운 목록에서 MS Excel 97-2000(데이터만)을 선택하고 내보내기 단추를 클릭하면 새 스프레드시트를 저장하거나 열 수 있습니다.

서식있는 텍스트 형식(RTF)으로 내보내기

내보내기 페이지의 드롭다운 목록에서 서식있는 텍스트 형식을 선택하고 내보내기 단추를 클릭하면 새 문서를 저장하거나 열 수 있습니다.

항목 목록으로

확대/축소

확대/축소 단추를 사용하면 보고서를 여러 크기(25% ~ 400%)로 볼 수 있습니다.

항목 목록으로

새로 고침

보고서를 새로 고치려면 새로 고침 단추를 클릭합니다.

항목 목록으로

인쇄

보고서 복사본을 인쇄하려면 인쇄 단추를 클릭합니다.

PDF 형식의 보고서가 나타나면 브라우저의 Adobe Acrobat Reader에서 인쇄를 선택합니다.

항목 목록으로

페이지별로 탐색

Crystal Reports에서와 마찬가지로 뷰어에서 "다음 페이지" 및 "이전 페이지" 단추를 사용하여 보고서를 탐색할 수 있습니다. 참고: 이 단추는 브라우저의 "뒤로" 및 "앞으로" 단추가 아닙니다. 보고서의 첫 페이지와 마지막 페이지로 이동할 수도 있습니다.

페이지 번호를 입력하고 "페이지로 이동" 단추를 클릭할 수도 있습니다.

항목 목록으로

검색 결과 테이블

보고서를 처음 표시하는 경우에는 검색 결과 테이블이 보고서의 맨 위에 나타납니다. 테이블 위에 있는 "검색 결과 표시/숨기기" 단추를 사용하여 결과 테이블을 숨길 수 있습니다.

테이블을 내보낼 수도 있습니다. 내보내면 Word 버전의 테이블이 만들어져 브라우저에 표시됩니다. 브라우저의 테이블을 전자 메일이나 Word 문서 또는 다른 문서로 복사할 수 있습니다.
